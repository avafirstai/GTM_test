# Playbook Semaines 2-4 — Scaling

## Semaine 2 — Iteration

### Objectifs
| Metrique | Objectif |
|----------|----------|
| Leads charges (cumule) | 1,000 |
| Emails envoyes/semaine | 400 |
| Open rate | 45-60% |
| Reply rate | 3-5% |
| Demos bookees | 3-5 |
| Premiers pilotes | 1 |

### Actions
1. **Analyser S1** — Quelle campagne performe ? Quels sujets ? Patterns dans les replies ?
2. **A/B tester** — Dupliquer la meilleure campagne, changer UNE variable
3. **Augmenter volume** — Si bounce < 3% et spam = 0 : passer a 40/inbox/jour
4. **Traiter replies** — Repondre < 2h, booker immediatement
5. **2 posts LinkedIn** — 1 resultat + 1 behind the scenes
